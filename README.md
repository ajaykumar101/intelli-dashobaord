# Smart Notifications Hub

## Problem
Important notifications from banks, utilities, and schools are scattered across emails and apps.

## Solution
A unified dashboard that analyzes incoming notifications and highlights what truly matters.

## MVP
- Email-based ingestion
- Bank & bill alerts
- Importance scoring
- Web dashboard

## Tech Stack
- Next.js
- Supabase Auth
- PostgreSQL
- LLM-based classification

## Roadmap
- Gmail integration
- Importance scoring
- Dashboard UI
